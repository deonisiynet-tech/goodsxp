## 🎯 ЛОГІКА РОБОТИ

### Завантаження пунктів видачі

```typescript
// ✅ Завантажуємо і відділення і поштомати одночасно
const loadWarehouses = async (cityRef: string) => {
  const response = await fetch("/api/nova-poshta/warehouses", {
    method: "POST",
    body: JSON.stringify({ cityRef }),
  });
  
  const result = await response.json();
  // result містить і відділення і поштомати
  setWarehouses(result);
};
```

### Визначення типу пункту

```typescript
// ✅ Тип визначається з даних API (TypeOfWarehouse)
const isPostomat = warehouse.type.toLowerCase().includes("почтомат");
```

### Відображення списку

```tsx
{warehouses.map((warehouse) => {
  const isPostomat = warehouse.type.toLowerCase().includes("почтомат");
  const icon = isPostomat ? "📦" : "🏢";
  const typeLabel = isPostomat ? "Поштомат" : "Відділення";
  
  return (
    <button key={warehouse.id}>
      <div className="flex items-start gap-3">
        <div className="text-xl">{icon}</div>
        <div>
          <div>{typeLabel} №{warehouse.number}</div>
          <div>📍 {warehouse.shortAddress}</div>
        </div>
      </div>
    </button>
  );
})}
```
