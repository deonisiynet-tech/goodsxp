## 🧪 ТЕСТИ ПЕРЕВІРКИ

### 1. Перевірка відображення товарів

1. Відкрийте `http://localhost:3000`
2. На головній сторінці мають бути "Популярні товари"
3. Перейдіть в `/catalog` - мають бути всі товари

### 2. Перевірка додавання товарів

1. Відкрийте `http://localhost:3000/admin/login`
2. Увійдіть як адміністратор
3. Перейдіть в `/admin/products`
4. Натисніть "Додати товар"
5. Заповніть форму:
   - Назва: "Тестовий товар"
   - Опис: "Опис тестового товару"
   - Ціна: 999
   - Залишок: 10
6. Натисніть "Зберегти"

**Очікуваний результат:** Товар створено, відображається в списку.

### 3. Перевірка відгуків

1. Відкрийте сторінку товару
2. Додайте відгук з рейтингом 1-5
3. Перевірте що рейтинг коректно відображається

## ⚠️ МОЖЛИВІ ПРОБЛЕМИ ТА РІШЕННЯ

### Помилка: "DATABASE_URL is not set"

**Рішення:** Перевірте `.env` файл в `server/`:
```env
DATABASE_URL="postgresql://postgres:password@localhost:5432/shop_db?schema=public"
```

### Помилка: "Port 5000 already in use"

**Рішення:** Змініть порт в `.env`:
```env
PORT=5001
```

### Помилка: "Prisma Client not generated"

**Рішення:**
```bash
cd server
npx prisma generate
```

### Товари не відображаються

**Рішення:**
1. Перевірте консоль сервера на помилки
2. Перевірте Browser Console на помилки
3. Переконайтесь що API доступне: `http://localhost:5000/api/products`

### Не працює завантаження зображень

**Рішення:**
1. Перевірте що папка `uploads/` існує
2. Перевірте Cloudinary credentials в `.env`
3. Або використовуйте локальне зберігання
