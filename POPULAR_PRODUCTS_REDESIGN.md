## 📝 ЗМІНИ

### 1. Кількість товарів: 5 замість 8

**Було:**
```tsx
<ProductList title="Популярні товари" limit={8} showAllLink />
```

**Стало:**
```tsx
<ProductList title="Популярні товари" limit={5} showAllLink popular />
```

---

### 2. Автоматична фільтрація популярних товарів

**Додано prop `popular`:**
```tsx
interface ProductListProps {
  title?: string;
  limit?: number;
  showAllLink?: boolean;
  popular?: boolean;  // ✅ Фільтр популярних товарів
}

// Фільтрація в loadProducts:
if (popular) {
  productsList = productsList.filter((p: Product) => p.isPopular === true);
  if (productsList.length > limit) {
    productsList = productsList.slice(0, limit);
  }
}
```

**Переваги:**
- ✅ Товари з админки з `isPopular: true` автоматично з'являються на головній
- ✅ Якщо популярних > 5, показуються перші 5
- ✅ Якщо популярних < 5, показуються всі

---

### 3. Розмір карток: зменшено для 5 товарів в ряд

**Було:**
```tsx
<div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 xl:grid-cols-4 gap-6">
```

**Стало:**
```tsx
<div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 xl:grid-cols-5 gap-6">
```

**Адаптивність:**
| Екран | Кількість колонок |
|-------|-------------------|
| Mobile (< 640px) | 1 колонка |
| Tablet (640-1024px) | 2 колонки |
| Desktop (1024-1280px) | 3 колонки |
| Large Desktop (> 1280px) | 5 колонок |

## 🎯 ТЕХНІЧНІ ДЕТАЛІ

### Логіка фільтрації:

```typescript
// 1. Отримуємо всі товари з API
const response = await productsApi.getAll({ limit });
let productsList = response.products || [];

// 2. Фільтруємо популярні
if (popular) {
  productsList = productsList.filter((p) => p.isPopular === true);
  
  // 3. Обмежуємо до 5
  if (productsList.length > limit) {
    productsList = productsList.slice(0, limit);
  }
}
```

### Админка (як позначити товар популярним):

В админці при редагуванні товару має бути чекбокс `isPopular`:

```typescript
// При створенні/оновленні товару:
{
  // ...
  isPopular: true,  // ✅ Товар з'явиться в блоці "Популярні"
  // ...
}
```
